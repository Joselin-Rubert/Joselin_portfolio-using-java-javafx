# Project Summary

## Overview

**Joselin's Premium JavaFX Portfolio Application** is a complete, production-ready desktop portfolio application built with Java and JavaFX. It showcases a professional portfolio with a dark futuristic theme, glassmorphism design, smooth animations, and interactive components.

## Features at a Glance

### ✨ Design & UX
- ⚫ Dark futuristic theme (black, purple, blue)

- ✨ Neon glow buttons with hover effects
- 🎨 Gradient backgrounds and accents
- 🌊 Smooth fade, slide, and scale animations
- 📱 Responsive desktop layout
