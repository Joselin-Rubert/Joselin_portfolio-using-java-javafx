## 📱 How to Use Documentation

### On Computer
1. Open files in your code editor
2. Use Ctrl+F (Cmd+F on Mac) to search
3. Click internal links to jump between docs

### In Terminal/Console
1. View files: `cat QUICK_START.md`
2. Search in file: `grep "phrase" README.md`
3. Count sections: `grep "^##" DEVELOPER_GUIDE.md | wc -l`

### In IDE
1. Open as separate tabs
2. Use IDE search (Ctrl+Shift+F)
3. Jump between docs using bookmarks
